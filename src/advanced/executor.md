# 执行器

## 通用参数

### `执行器类型`

- 类型：`enum{施力, 播放音频, 设置重生点, 玩家死亡, 游戏结束, 完成关卡, 跟随路径, 断开关节}`
- 默认值：`施力`

### `后继执行器编号`

- 类型：`int`
- 默认值：`0`

如果前一执行器为瞬时执行器，后一执行器将同时开始执行；否则后一执行器在前一执行器完成后开始执行。

## 施力

### `目标编号`

- 类型：`Tuple<int, int[]>`
- 默认值：`(0, [])`
- 限制：`(n, l): n >= 0`

开始执行时，这些物体会受力 **（无论它们在哪里）**。

### `施力模式`

- 类型：`enum{附加恒力, 设定速度, 附加加速度}`
- 默认值：`附加加速度`

| 施力模式   | 触发器类型为停留时 | 否则         |
| ---------- | ------------------ | ------------ |
| 附加恒力   | 恒力场             | 恒冲量发生器 |
| 设定速度   | 恒速器             | 瞬时变速器   |
| 附加加速度 | 恒加速度场         | 恒速度附加器 |

### `受力方向`

- 类型：`enum{朝向 Y 轴, 朝向中心}`
- 默认值：`朝向 Y 轴`

### `力度`

- 类型：`float`
- 默认值：`0`

负数表示反方向。

### `使用衰减`

- 类型：`bool`
- 默认值：`false`

勾选此项时，物体离触发器越远受力越小。

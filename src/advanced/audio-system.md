# 音频系统

`创建 → 音频播放器`

## 参数

### `音频文件`

- 类型：`Audio`
- 默认值：空

### `作为背景音乐`

- 类型：`bool`
- 默认值：`false`

### `循环`

- 类型：`bool`
- 默认值：`false`

### `音量`

- 类型：`float`
- 默认值：`1`
- 限制：`v: 0 <= v <= 1`

### `空间音频` <badge text="作为背景音乐 = false"/>

- 类型：`bool`
- 默认值：`false`

空间音频具有立体声和多普勒效应。

### `最小距离` <badge text="空间音频 = true"/>

- 类型：`float`
- 默认值：`1`
- 限制：`d: 0 <= d <= 100`

摄像机与音源距离大于 `最小距离` 时，音量开始衰减。

### `最大距离` <badge text="空间音频 = true"/>

- 类型：`float`
- 默认值：`15`
- 限制：`d: 0 <= d <= 100`

摄像机与音源距离大于 `最大距离` 时，无法听到声音。

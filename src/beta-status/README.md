## 破坏性更新

### `0.9.5b1`

- 关卡结算公式改为 `最终分数 = 时间分数 + 耐久度 * 5`

### `0.9.2b1`

- 调整耐久相关机制
  - 耐久初始值为 100，上限为 200
  - 耐久大于 100 的部分体现为深色条，该部分不提供受伤缓冲（浅色条），且恢复效果为原先的 20%
    - 例如，在耐久为 90 时恢复 100 耐久，耐久将变为 `118 = 90 + 10 + 90 * 0.2`
  - 激活盘点时，若耐久不足 100 则补足到 100
  - 耐久加入关卡结算~~，公式为 `最终分数 = 时间分数 + 耐久度 * 10`~~
